# Proyecto Informativo: Arquitecturas Web Modernas
**Desarrollador:** Luis Pablo Mendoza Saucedo
**Institución:** Universidad Autónoma de Aguascalientes (UAA)
**Materia:** Desarrollo Web

## Descripción
Este sitio web informativo explora las tecnologías de vanguardia en el desarrollo del lado del servidor y del cliente, incluyendo Web APIs (REST/SOAP), Microservicios, Contenedores (Docker/Kubernetes) y Seguridad con JWT.

## Tecnologías Utilizadas
- **Frontend:** HTML5, CSS3, JavaScript (ES6).
- **Framework de Diseño:** Bootstrap 5.
- **Contenidos:** Basados en materiales de IBM, Google Cloud y AWS.

## Requisitos de la Rúbrica Cumplidos
- Diseño homogéneo y moderno.
- Formulario con validación de 5 campos en JS.
- Uso de componentes: Navbar, Carrusel, Cards, Modales y Acordeones.
- Material multimedia: Videos de YouTube y Mapa de ubicación.
